Idepix Landsat-8 OLI plugin

The Idepix Processor provides a pixel classification into properties such as clear/cloudy, land/water, snow, ice etc. The processing options/parameters as well as the underlying classification algorithms are instrument-dependent. Actually, SNAP Idepix plugins for the following satellites/sensors are available:

Satellite Sensor
Sentinel-2 MSI
Sentinel-3 OLCI
Envisat MERIS
Landsat-8 OLI
Proba-V Vegetation
SPOT Vegetation
Terra/Aqua MODIS
OrbView-2 SeaWiFS
Suomi NPP VIIRS
