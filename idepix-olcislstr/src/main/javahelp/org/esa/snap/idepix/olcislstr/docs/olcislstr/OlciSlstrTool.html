IdePix Sentinel-3 OLCI

NOTE: This operator uses TensorFlow to execute neural nets. TensorFlow has some special requirements to the system. The CPU needs to support the AVX instruction set (available since 2011 Intel CPU SandyBridge) and also the Microsoft Visual C++ 2015 Redistributable needs to be installed. The AVX instruction set is usually no problem, only when you are using a virtual machine then it can be, that the instruction set needs to be enabled explicitly.
